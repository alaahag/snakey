# Snakey (Game)

A full-stack MATRIX JavaScript game.

**Technology Stack**: Express, Mongoose, Node & jQuery.

```
Supports Solo and 1vs1 play.

Supports changing difficulty (speed) of the game, and top online scores for each difficulty.
```
